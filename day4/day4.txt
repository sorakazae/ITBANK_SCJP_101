하지만 인스턴스와 객체는 거의 동일한 용어.

new : 메모리 할당, 객체 생성

레퍼런스 카운터가 1 이상이면 가비지(쓰레기)가 아님

class확장자 = 자바(전용) 실행 파일 (JVM필요)

class는 여러개 만들 수 있다.

어디서든지 엑세스 하려면 public을 써준다.

public class 는 그 소스(.java)와 이름이 같아야만 가능(단 하나)

재귀호출 -> 반복문   가능

main메서드 = 기본메서드
main메서드에는 static이 있어야 한다.

메서드 안에 선언되는 변수    지역변수 (메서드 안에서만 사용 가능) 
메서드 수행이 끝나면 사라짐

메서드 밖에 선언되는 변수    맴버변수
          static이 붙은 맴버변수           클래스변수    클래스 로딩할 때 생성 (한번만 할당)
                      모든 객체들이 공유
          static이 붙지 않은 맴버변수   인스턴스변수 인스턴스가 생성될때   (여러번 할당)
                      모든 객체가 각각 사용
